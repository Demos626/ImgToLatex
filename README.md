# ImgToLatex
Приложение для перевода мат. формулы с картинки в Latex строку
- Вырежи формулу
- Сохрани
- Выбери изображение
- Настрой бинаризацию
- Получи формулу

# Поддерживает сейчас
- Простые дроби
- Индексы и степени
- Индексы и степени в простых дробях

# Ближайшие планы

- Улучшить бинаризацию
- Добавить вывод рамок для визуального восприятия опроделения символов
